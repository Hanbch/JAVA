1. 아래가 의도하지 않는 결과를 출력한다 이유와
고치는 방법을 설명하시오.


		int num1 = 3;
		int num2 = 4;

		double result = num1 / num2;
		
		System.out.println(result); //0.0

 - 위의 결과 값은 0.0 이다. 이유는 변수 num1,2가 int 로 선언되어있다.
 그러므로 연산시 결과값이 정수로 나온다. 연산 결과가 0.75이지만 실수부분은 버 리고 정수 0 만 나오게된다. 0이라는 결과값을 double, 실수형 데이터 타입에 담았 기 떄문에 result의값은 0.0이된다. 0.75가 출력이되게 하려면 num1,2를 double   형 으로 선언하거나 연산할때 형변환(casting)을 해준다.  
 ex) double result = (double)num1 / num2;
      double result = num1 / (double)num2;
      double result = (double)num1 / (double)num2;

2. 상수란 무엇이며 변수와의 차이는?
 - 상수란 변하지 않는 수 이다. 변수는 초기화한뒤 값을 바꿀수있지만. 
상수는 한번 선언하고 변경 할 수 없다. 상수는 선언할때 데이터타입 앞에 final 붙여서 선언한다. 

3. 상수를 이름 짓는 방법은?
 - 통상적으로 상수명은 모두대문자로 하며 단어 사이에 언더바를 사용해 상수명만 봐도 상수인것을 알수있게 명명한다. ex) final int MY_AGE = 20;

4. 초기화란 무엇인가? 
 - 초기화란 변수를 선언하고 처음 값을 할당시켜주는 것이다.
 선언 int num;
 초기화 num = 0; 

5. 리터럴이란 무엇인가요?
 - 연산이 들어가있는 변수에서 각 숫자들을 리터럴이라고 한다. 
 예를들어 int num = 1 + 2;  num에는 1 + 2의 연산 결과인 3이 저장되지만 
 메모리에는 1 과 2 도 저장이되어서 cpu 로 보내주게 된다. 이때 1 과 2 를 리터럴 이라고 한다.  

6. 아래의 내용이 에러가 나는 이유와 수정 방법은?
System.out.println(3147483647 + 3147483648); //에러남

 - java의 정수형 데이터타입의 디폴트값이 int 인데 3147483647이 int 범위를 넘어 서 에러가 난다. 그러므로 int 범위밖의 숫자에는 뒤에 L을 붙여 데이터 타입은 long으로 바꿔줘야된다. System.out.println(3147483647L + 3147483648L);

7.자동 형변환과 명시적 형변환에 대하여 설명하시오.
 - 자동 형변환은 다른 두 데이터 타입이 연산될때 명시적 형병환을 하지 않아도 
  자동으로 작은데이터 타입이 큰데이터 타입으로 변환되고 정수데이터 타입이 
  소수 데이터 타입으로 변환되는것 을 말한다.
  ex) int a = 1;
       double b = 2;
       System.out.print(a + b);  // 2.0  	

  명시적 형변환은 변수 앞에 변환할 데이터 형을 명시해주므로써 데이터 형을 
  변환 해주는 것이다. 그러나 큰 데이터 형에서 작은 데이터 형으로 변환시 값이 
  변형 될 수 있다. 또한 실수에서 정수로 변환시 소수점아래수는 생략된다. 
  ex) double a = 1.23;
      int b = (int)a;  // 1
