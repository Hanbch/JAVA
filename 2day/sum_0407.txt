1.Hello World 를 커맨트 창으로 찍으시오.

메모장에 코드입력 -> class명.java로 파일저장 ->  명령프롬프트 켜고 파일저장된 디렉토리로접근(cd 경로) - >
dir 입력후 HelloWorld.java 파일확인 -> javac HelloWorld.java 입력(컴파일 되면서 Hello.class 생성됨) - dir/w입력후 디렉토리 확인 ->
java HelloWorld 입력( 출력 )

2. 아래의 명령어를 설명하시오.(뭐 하는 건가?)
-javac 명령어 : java에서 작업한 소스를 바이트코드로 컴파일해주는 명령어
-java 명령어 : 컴파일된 .class 파일을 실행해주는 명령어

3. 컴파일이란 무엇인가요?
프로그래밍 언어 -> 기계어(2진수) 로 변환

4. java 언어를 창시한 사람은?
제임스 고슬링

5. JDK란 무엇이며, 어디서 다운로드 받으며, OS별로 버전이 있는 까닭은?
JDK 란 Java developer’s kit 자바 개발 도구이다.
OS 마다 버전이있는 이유는 리눅스, MAC, 윈도우 등 운영체제 마다 다른버전이 필요하기 때문이다.
